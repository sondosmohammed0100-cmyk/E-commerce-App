🛒 E-Commerce API
A RESTful API for an e-commerce application built with Node.js, Express, and MongoDB.
It supports authentication, role-based authorization, products & categories management, image uploads, and pagination.

 Project Structure
project/
│
├── Controller/
├── Models/
├── Router/
├── Middelware/
├── Validation/
├── utils/
├── Uploads/
├── app.js
└── .env
⚙️ Installation
git clone <repo-url>
cd project
npm install
Run server
npm run dev
