🛒 E-Commerce API
A RESTful API for an e-commerce application built with Node.js, Express, and MongoDB.
It supports authentication, role-based authorization, products & categories management, image uploads, and pagination.
🚀 Features
🔐 User Authentication (Register / Login)
🛡 Role-based Authorization (Admin / User)
📦 CRUD operations for Products
🗂 CRUD operations for Categories
🖼 Image Upload using Multer
📄 Pagination for products
🔍 Populate relations (category & user)
⚠️ Centralized error handling
✅ Input validation using Joi
🏗 Tech Stack
Node.js
Express.js
MongoDB (Mongoose)
JWT Authentication
Multer
Joi
Morgan
CORS
